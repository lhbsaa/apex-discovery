# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.2.x   | ✅ Active development |

## Reporting a Vulnerability

apex-discovery is a development tool that runs locally with zero network dependencies.
It does not collect telemetry or send data externally.

If you discover a security vulnerability, please open an issue at:
https://github.com/lhbsaa/apex-discovery/issues

We will review and respond within 48 hours.
